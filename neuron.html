Neuron Inventor Kit

The Makeblock Neuron Inventor Kit, an essential package including programmable electronic building blocks of sound, light, sensors, etc. Combined with intuitive flow-based programming, Neuron encourage all creators to turn unlimited ideas into real gadgets.

Key Features

Simple & Durable

Building blocks are connected in a magnetic snap with Pogo Pin, with the added benefit of making each block more durable.

Graphical Programming

Graphical & flow-based programming in Neuron app enables beginners to easily handle Neuron application and perform basic to advanced functionality (e.g. IoT) without prior coding experience.

Widely Extensible

Compatible with Makeblock platform and LEGO® blocks, fun is infinite with great ideas. Neuron gadgets can be controlled and/or programmed using mBlock& Neuron App.
